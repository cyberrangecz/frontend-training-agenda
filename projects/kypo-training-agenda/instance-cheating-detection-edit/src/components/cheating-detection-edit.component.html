Cheating Detection Methods
Answer Similarity
Shows users that submitted a flag that belongs to someone else. Don't use for trainings without variant answers
Location Similarity
Shows users that had similar IP in training.
Time Proximity
Shows users submitting answers in a similar time frame. Specify the proximity in the frame below.
Minimal Solve Time
Shows users that solved a task in less than minimal possible solve time.
No Commands
Shows users that used no commands to solve a level requiring commands.
Forbidden Commands
Shows users that submitted forbidden commands. Specify the commands in a window below.
